2.substring() 和 slice()

substring() 和 slice() 方法返回索引中指定的字符串。
substring() 和 slice() 这两个方法功能其实是一样的，包头不包尾。
但是区别是 slice() 可以传入负数，substring() 不接受负数。
slice() 传入负数，-1代表最后一个元素，-2代表倒数第二个元素，以此类推。
